# Claude Phase 9 战役 - 第二回合：分享卡片底座生成

干得漂亮！深色模式的状态流转已经完美生效。接下来进入我们 Phase 9 的第二个杀手级功能：**沉浸式跨平台分享卡片 (Share Card Export)**。

Gemini 已经在全局 CSS 中提前埋好了分享卡片的样式类名（`.share-card-wrapper`, `.share-card-header`, `.share-card-body`, `.share-card-footer` 等）。你的任务是搭建 React 骨架与 Canvas 的渲染逻辑。

### 任务 1：安装底层绘图依赖
在项目根目录执行：
`npm install dom-to-image-more file-saver`
*(之所以不用 html2canvas 是因为它对现代 CSS 如 backdrop-filter 的支持不够完美，dom-to-image-more 会有更真实的还原度)*

### 任务 2：创建隐藏的 `<ShareCard>` 组件
新建文件 `src/components/ShareCard.jsx`。这个组件由于是用来截图的，它**绝对不能**干扰当前屏幕的阅读。
请按照以下 DOM 结构和类名来编写它（必须接受 `diary` 和 `pomodoroSum` 作为 props）：
```jsx
// src/components/ShareCard.jsx
import React, { forwardRef } from 'react';
import { marked } from 'marked';
import DOMPurify from 'dompurify';

const ShareCard = forwardRef(({ diary, pomodoros = [] }, ref) => {
  // 计算今天总专注时长 (分钟)
  const totalFocusMinutes = pomodoros.reduce((total, p) => total + (p.duration / 60), 0);
  
  // 安全渲染 Markdown
  const renderMarkdown = (content) => {
    return { __html: DOMPurify.sanitize(marked(content || '今天没有记录任何感悟。')) };
  };

  return (
    <div style={{ position: 'absolute', left: '-9999px', top: 0 }}>
      {/* 必须给 ref 绑定到真正用来截图的 wrapper 上，同时给定一个固定的宽度，比如 800px，保证生成出的图片比例完美 */}
      <div ref={ref} className="share-card-wrapper" style={{ width: '800px', padding: '40px', background: 'var(--bg-primary)' }}>
        
        <div className="share-card-header">
          <h2>{diary?.title || '未命名日记'}</h2>
          <div className="share-card-date">{diary?.date}</div>
        </div>

        <div className="share-card-body markdown-body" 
             dangerouslySetInnerHTML={renderMarkdown(diary?.content)} />

        <div className="share-card-footer">
          <div className="footer-stats">
            🍅 今日专注: <strong>{Math.round(totalFocusMinutes)}</strong> 分钟
          </div>
          <div className="footer-brand">
            — Generated by MindDiary
          </div>
        </div>

      </div>
    </div>
  );
});

export default ShareCard;
```

### 任务 3：在写日记的主面板注入【分享】按钮逻辑
在现有的 `src/components/Welcome.jsx` (或者是日记主渲染区)：
1. 引入 `dom-to-image-more` 和 `file-saver`。
2. 引入 `<ShareCard>` 并在组件 return 的最末尾挂载它，传入当前的 `selectedDiary` 和对应的按日期拉取的专注记录（可以从 DiaryContext 里顺便补充一个 `getPomodorosByDate` 的接口，或者直接从 localDB 里拉取当天的番茄钟数据）。
3. 在日记编辑器标题区域或者侧边栏工具区增加一个按钮：`🖼️ 生成今日分享图`。
4. 点击该按钮时，利用 `useRef` 获取到 `<ShareCard>` 的节点，调用 `domToImage.toBlob(node, { scale: 2 })` (设置 scale=2 可以输出超高清图片)，并用 `saveAs(blob, 'MindDiary-Share.png')` 触发下载。

完成后，点击【生成今日分享图】测试是否能下载出一张 `800px` 宽度的 PNG 图片。如果成功，请向指令长 (User) 汇报。
